🚀 Sunucuyu Nasıl Yayına Alırsın?

Aşağıdaki seçeneklerden birini kullan:

🆓 Seçenek 1: Railway (Ücretsiz, Önerilen)
  1. railway.app'e git ve GitHub ile giriş yap
  2. "New Project" → "Deploy from GitHub repo" seç
  3. Sunucu klasörünü GitHub'a yükle (ya da ZIP'i sürükle bırak)
  4. Variables (Ortam Değişkenleri) bölümüne şunları ekle:
ADMIN_PASSWORD=güçlü_şifren
JWT_SECRET=rastgele_uzun_bir_string
LICENSE_SECRET=başka_bir_rastgele_string
5. Deploy olunca Railway sana bir URL verir (örn: mrtbot-xxx.railway.app)
🖥️ Seçenek 2: Kendi VPS'in (Ubuntu)
# Node.js kur
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt install -y nodejs

# Sunucu dosyalarını yükle ve çalıştır
cd mrtbot-server
cp .env.example .env
nano .env # şifreleri değiştir
npm install
npm install -g pm2
pm2 start server.js --name mrtbot
pm2 save
🔌 Sunucu URL'ini Bot'a Tanımla

Sunucu çalıştıktan sonra background.js içindeki şu satırları güncelle:

// ESKİ (fociisoftware.com):
const urls = ["https://glad.fociisoftware.com/validate-token", ...]

// YENİ (kendi sunucun):
const urls = ["https://SENİN-SUNUCUN.railway.app/validate-token"]
